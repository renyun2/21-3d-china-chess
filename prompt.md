# 项目 21：3D 多人在线中国象棋

> 本文件仅描述需求，不包含任何实现代码。

## 一、项目简介
3D 渲染的中国象棋平台，支持：好友对战、AI 对战（Pikafish.wasm 多档难度）、观战、复盘。整站为 Web，鼠标拖拽走子，棋子动画顺滑。

## 二、技术栈
- 构建：Vite + TypeScript
- 框架：React 18
- 3D：Three.js + @react-three/fiber + drei
- 棋盘逻辑：xiangqi.js（或等价的中国象棋规则库）
- AI：Pikafish.wasm（Web Worker 内运行）
- 网络：Socket.IO（Node.js + TypeScript）
- 持久化：SQLite + Prisma（对局历史）
- 鉴权：JWT + bcrypt
- UI：TailwindCSS + shadcn/ui
- 动画：GSAP

## 三、核心功能需求
1. **3D 棋盘**
   - 9×10 标准中国象棋棋盘（楚河汉界、九宫格）
   - 自定义 PBR 木纹/大理石材质
   - 可切换棋子风格（经典/极简/卡通）
   - 摄像机预设：俯视、侧视、第一视角（FPV）
2. **走子**
   - 鼠标拖拽 + 高亮可走格
   - 走子动画 + 提子动画
   - 规则校验：蹩马腿、塞象眼、炮架、兵卒过河前后走法、将帅不能照面（飞将）
   - 将军 / 应将提示
3. **对战模式**
   - 在线匹配（按 Elo 范围）
   - 创建房间 / 加入房间（链接邀请）
   - AI 对战：1-20 档难度（Pikafish depth）
4. **比赛规则**
   - 计时：闪电（3+0）/快棋（5+5）/慢棋（15+10）
   - 认输、提和
   - 和棋：双方同意、60 回合无吃子、三次重复局面
   - 判负：长将、长捉（按中国象棋竞赛规则处理）
5. **复盘**
   - 棋谱导入/导出（ICCS 或中文记谱法）
   - 引擎评估走法（每步标记 ?/!/!! 等）
6. **社交**
   - 聊天室、好友、对局历史
   - 排行榜 + Elo 积分
7. **观战**
   - 进行中对局列表
   - 实时同步走子

## 四、编译与调试
- `pnpm dev` 同时启动 client + server
- 同浏览器开两个隐身窗口测试匹配
- `pnpm build` 输出可部署产物
